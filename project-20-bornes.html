Refonte du portail web "Les 20 Bornes d'Andrésy"

Modernisation de l'expérience utilisateur et intégration responsive pour un événement sportif local.

Rôle

Designer UI & Intégrateur Front-End

Outils

HTML5, CSS3, JS (Vanilla), Bootstrap 5, Figma

Le Contexte

L'objectif était de créer une interface dynamique et informative pour la 43ème édition des 20 Bornes d'Andrésy. Le site devait capter l'énergie de l'événement tout en permettant aux utilisateurs de s'inscrire rapidement et de trouver les informations techniques (parcours, horaires, règlement) sans friction, quel que soit leur appareil (mobile ou desktop).

Identité Visuelle & Émotion

J'ai opté pour une palette de couleurs vibrantes (Magenta, Orange, Sarcelle) pour refléter le dynamisme du running et l'aspect festif. L'utilisation de formes organiques animées (morphing blobs en CSS) en arrière-plan brise la rigidité de la grille pour apporter du mouvement.

Hiérarchie de l'information

L'information cruciale est segmentée. Les 3 distances (21km, 10km, 5km) sont présentées sous forme de cartes distinctes avec des codes couleurs propres pour faciliter la prise de décision rapide de l'utilisateur.

Call-to-Action (CTA)

Le bouton 'S'inscrire' est placé stratégiquement dans la zone de flottaison (Hero Section) et se distingue par un fort contraste pour maximiser le taux de conversion.

Accessibilité & Lisibilité

Choix des typographies Montserrat (Titres) pour l'impact et Roboto (Corps) pour la lisibilité optimale des textes longs.

Développement & Intégration

Maîtrise du Framework & Customisation

J'ai utilisé Bootstrap 5.3 pour assurer une grille responsive robuste, mais j'ai surchargé le design (overriding) via un fichier style.css dédié et l'usage de variables CSS (:root) pour ne pas ressembler à un site Bootstrap standard.

CSS Moderne & Animations

Intégration d'animations CSS avancées (@keyframes pour le morphing des formes et les effets de pulsation) pour enrichir l'interface sans alourdir le chargement avec des librairies externes. Utilisation de backdrop-filter et de dégradés pour des effets de profondeur modernes.

JavaScript & Performance

J'ai développé un script 'Back to Top' natif sans jQuery. J'ai optimisé l'animation de défilement en utilisant requestAnimationFrame plutôt qu'un simple setInterval pour garantir une fluidité parfaite (60fps), notamment sur mobile où les ressources sont limitées.

Responsive Design

Approche Mobile-First. Le menu de navigation se transforme en menu burger sur mobile avec une animation soignée, et les cartes s'empilent naturellement grâce aux classes de grille Bootstrap (col-md-6, col-lg-4).
